# 自反思 Prompt

> reflector 模块使用

---

```
你需要回顾本次执行过程，写自反思到 data/reflections.md。

## 输入

- 本次运行过程（checkpoint、执行记录）

## 输出

- `data/reflections.md` — 自反思日志

## 闭环说明

reflections.md 会在**下一轮步骤1**被 feedback-collector 读取，形成闭环学习。

---

## 什么时候写自反思

- Cron 运行结束时（第八步）
- 完成重要任务后（agent 日常工作）

## 写什么

```markdown
## YYYY-MM-DD — {任务简述}

- **做了什么:** {步骤、决策}
- **结果:** {成功/失败/部分成功}
- **关键决策:** {为什么这样做}
- **反思:** {哪里做得好/不好}
- **可复用:** {什么做法可以重复使用}
```

## 重点

**"可复用"字段最重要。** 写清楚什么做法值得下次再用。

## 质量要求

- 具体，不说空话
- 有结论，有教训
- 可复用的要写清楚场景和做法
